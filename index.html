Phishing Awareness Simulator

Cyber Security Internship Project

Project Overview

This project demonstrates a phishing awareness simulation designed to educate users about common phishing techniques and improve cybersecurity awareness.

Objectives

  • Identify phishing indicators.
  • Recognize suspicious emails and links.
  • Understand social engineering tactics.
  • Improve cybersecurity awareness.
  • Reduce phishing-related risks.

Simulation Scenarios

  • Fake Banking Email
  • Password Reset Scam
  • Prize Winner Scam
  • Fake Login Page
  • Business Email Compromise

Warning Signs of Phishing

  • Suspicious sender address
  • Urgent requests for action
  • Unexpected attachments
  • Misspelled URLs
  • Requests for sensitive information

Prevention Recommendations

  • Verify sender identities.
  • Enable Multi-Factor Authentication.
  • Never click unknown links.
  • Check URLs before logging in.
  • Report suspicious emails.
